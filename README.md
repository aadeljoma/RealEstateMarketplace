# Real Estate Marketplace
Build a dapp for real state marketplace using ERC721 , OpenSea framework and Zokrates framework.

## Setup

`npm install`

`truffle compile`

`truffle migrate --network rinkeby`

## Tests

`cd eth-contracts`
`truffle test`
